# 🚀 Быстрый старт

## 📱 Веб-сайт визитка

**Готов к использованию!** Просто откройте `index.html` в браузере или разместите на любом веб-хостинге.

### GitHub Pages (рекомендуется)
1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта
3. Включите GitHub Pages в настройках
4. Ваш сайт будет доступен по адресу: `https://username.github.io/repository-name`

## 🤖 Telegram боты

### Шаг 1: Получите токены ботов
1. Найдите @BotFather в Telegram
2. Создайте 5 ботов командой `/newbot`:
   - Бот покупки недвижимости
   - Бот продажи недвижимости  
   - Бот инвестиций
   - Бот страхования
   - Бот оценки
3. Сохраните полученные токены

### Шаг 2: Настройте окружение
```bash
# Скопируйте пример конфигурации
cp .env.example .env

# Отредактируйте .env файл, добавив ваши токены:
BOT_P_TOKEN=ваш_токен_бота_покупки
BOT_PR_TOKEN=ваш_токен_бота_продажи
BOT_INV_TOKEN=ваш_токен_бота_инвестиций
BOT_STR_TOKEN=ваш_токен_бота_страхования
BOT_OCENKA_TOKEN=ваш_токен_бота_оценки
```

### Шаг 3: Запустите боты

#### Вариант A: Docker (рекомендуется)
```bash
# Запуск всех ботов
docker-compose up -d

# Проверка статуса
docker-compose ps
```

#### Вариант B: Python
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск конкретного бота
python BOT_Inv.py
```


## ✅ Проверка работы

1. **Веб-сайт**: Откройте `index.html` - должна отображаться визитка Дмитрия
2. **Боты**: Найдите ваших ботов в Telegram и отправьте команду `/start`
3. **Ссылки**: Проверьте, что кнопки на сайте ведут к правильным ботам
